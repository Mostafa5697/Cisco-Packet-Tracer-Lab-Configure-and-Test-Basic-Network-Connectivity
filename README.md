# Cisco-Packet-Tracer-Lab-Configure-and-Test-Basic-Network-Connectivity

A hands-on Cisco Packet Tracer project covering the end-to-end configuration of a small office LAN. This project spans six lab videos and walks through switch setup, static and dynamic IP addressing, router interface activation, Layer 2 switch management via SVI, Telnet remote access, and dual-pool DHCP configuration for wired and wireless clients.

## Topics Covered
- Cisco IOS CLI navigation and switch/router configuration
- Running-config vs. startup-config and configuration persistence
- Static IPv4 addressing (PCs, Laptop, Server, Router interface)
- VLAN 1 SVI configuration for Layer 2 switch management
- VTY Telnet access with password and enable secret
- DHCP pool configuration for wired and wireless subnets
- Ping-based connectivity testing and physical fault simulation

## Technologies
Cisco Packet Tracer | Cisco IOS | Cisco 2960-24TT | Cisco ISR4331 | IPv4 | DHCP | Telnet | VLAN SVI

## Relevance
Demonstrates practical ability to configure, secure, and verify a multi-device network from scratch.
